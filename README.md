Приложение для пересылки всех SMS сообщений с телефона в Telegram бота

APK доступно в релизах

Для установки необходимо [отключить Play Защиту](https://support.google.com/accounts/answer/2812853?hl=ru)
Также нужно [снять ограничение](https://support.google.com/android/answer/12623953?visit_id=638990036956655314-3358318425&p=restricted_settings&rd=1) на включение разрешения для получения SMS
